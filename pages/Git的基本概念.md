# 版本库
	- git版本库就是一个简单的数据库，包含了用户所修改版本和提交的版本以及历史变更的信息，而且还提供了版本库本身的副本。
	- git在每个版本库里维护了一组该版本库的配置值，如果你克隆或者复制了该版本库，那么配置值是根据你设定的值来执行，比如e
	- asdfmail等信息。
# Git的对象类型
	- **块（blob）**
	  文件的每一个版本表示为一个块(blob)。
	- **目录树（tree） **
	  一个目录树（tree）对象代表一层目录信息。
	- **提交（commit）**
	  一个提交（commit）对象保存版本库中每一次变化的元数据，包括作者、提交者、提交日期和日志消息。
	- **标签（tag）**
	  一个标签对象分配一个任意的且人类可读的名字给一个特定对象，通常是一个提交对象。
- # 索引
	- 索引是一个临时的、动态的二进制文件，它描述整个版本库的目录结构。
